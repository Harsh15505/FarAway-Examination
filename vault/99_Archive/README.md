## Purpose

This directory stores archived documents that are no longer active but may be referenced for historical context.

---

## Archive Policy

Move documents here when:
- A sprint is completed and its board is superseded
- An ADR is superseded by a newer decision
- A feature is deprecated or removed
- A document is replaced by a more comprehensive version

---

## Naming Convention

Prefix archived files with the archive date: `YYYY-MM-DD_OriginalName.md`
