⚠️ What to Know Before Testing Phase 2b:
The backend runs in SQLite mode (no Docker needed). Most Phase 2b pages show a yellow demo data banner because the backend stubs for create/list exams are not fully wired up yet. This is expected behaviour — do not mark these as failures.

Mark as PASS if: demo data shows, the banner appears, modals open correctly, and buttons fire network requests (check browser DevTools Network tab).
